# Web Development Assignment Portal

## 🚀 Quick Start

## Vercel Deployed link
```bash
web-development-assignment-1-lac.vercel.app
```

### Start Python Server
```bash
python -m http.server 8000
```

### Start PHP Server
```bash
php -S localhost:8001
```

### Open Project
Visit: `http://localhost:8000`

## 🎯 How to Use

1. Select category from main page
2. Click problem to view code
3. Edit and see results in real-time
4. Click "Reset Code" to restore original
5. Use "Back" button to return

## 🔧 Servers

| Server | Port | Purpose |
|--------|------|---------|
| Python | 8000 | HTML, CSS, JavaScript, React |
| PHP | 8001 | PHP problems |
