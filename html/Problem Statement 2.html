Personal Computer System

Monitor

The monitor is an output device that displays text, images, and videos generated by the computer.

External Speaker

Speakers are output devices used to produce sound such as music, voice, and system alerts.

Keyboard

The keyboard is an input device used to enter text, numbers, and commands into the computer.

Computer Tower (CPU Cabinet)

The computer tower contains the CPU, motherboard, RAM, storage devices, and power supply.

Mouse

The mouse is a pointing input device used to click, scroll, and interact with graphical elements.

Wireless Internet Router

The wireless router connects the computer to the internet and enables network communication.
